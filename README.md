# Zerodha Clone - Full Stack Trading Platform

## 📌 Project Description

Zerodha Clone is a full-stack trading platform inspired by Zerodha Kite. It provides a trading dashboard experience where users can manage watchlists, view holdings, track positions, and place orders. The project is built using modern web technologies with a React-based frontend, Node.js backend, and MongoDB database.

## ✨ Features

- Responsive trading interface
- Watchlist management
- View stock holdings
- View trading positions
- Place buy orders
- Order management system
- Portfolio summary
- Real-time data updates using API integration
- Backend REST API integration
- Database management with MongoDB
